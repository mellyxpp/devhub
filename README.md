# DevHub – Code. Lerne. Wachse.

Willkommen bei **DevHub**, deiner interaktiven Lernplattform für Programmierung! Hier kannst du direkt im Browser die Grundlagen verschiedener Programmiersprachen üben und deinen Fortschritt verfolgen.

## 🔗 DevHub sofort ausprobieren

## 📖 Über das Projekt

DevHub ist eine browserbasierte Lernplattform, die angehenden Entwicklern den Einstieg in die Programmierung erleichtert. Die Idee entstand aus dem Wunsch, eine niedrigschwellige Möglichkeit zum Üben zu schaffen – ganz ohne komplexe Entwicklungsumgebungen. Mit einem interaktiven Split-Screen-Design siehst du links die Aufgabenstellung und rechts den Code-Editor. Dein Fortschritt wird automatisch gespeichert, und der kleine Helfer "Bluebot" gibt dir hilfreiche Tipps.

## ✨ Features im Überblick

*   **5 Programmiersprachen**: Python, JavaScript, Java, HTML/CSS und C++
*   **5 Grundkonzepte pro Sprache**: Variablen, Bedingungen, Schleifen, Listen, Funktionen
*   **Integrierter Code-Editor**: Schreibe und führe Code direkt im Browser aus
*   **Intelligente Prüfung**: Deine Lösungen werden auf die wichtigsten Schlüsselkonzepte geprüft
*   **Fortschrittsspeicherung**: Deine Erfolge und Notizen bleiben erhalten
*   **Bluebot-Assistent**: Er gibt dir Tipps und motiviert dich

## 🛠️ Verwendete Technologien

*   HTML5
*   CSS3 (Flexbox/Grid für responsives Design)
*   JavaScript (Vanilla)
*   LocalStorage API

## 🚀 Installation & Nutzung

So kannst du DevHub lokal nutzen oder anpassen:

1. **Repository klonen:**
   ```bash
   git clone https://github.com/mellyxpp/devhub.git
   ```

2. **Öffnen:** Navigiere in den Ordner und öffne `devhub.html` in einem beliebigen Browser.

3. **Lernen:** Wähle eine Sprache, klicke auf eine Aufgabe und leg los!
